## How to use

-  Download & install Docker & Docker compose.

-  Set folder in working directory.

-  Execute init file.

    ~~~sh
    source dockerfile_init.sh
    ~~~

- Execute boot file.

    ~~~sh
    source dockerfile_boot.sh
    ~~~

-  If you want to stop, execute stop file.

    ~~~sh
    source dockerfile_stop.sh
    ~~~
